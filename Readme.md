## Event Planner App

The Event Planner App is an Android application developed to help users efficiently manage their daily events. The app allows users to create, view, update, and delete events, ensuring a smooth and organized scheduling experience.

The application uses the Room Persistence Library for local data storage, which ensures that all event data is saved even after the app is closed or the device is restarted. A RecyclerView is implemented to display the list of upcoming events dynamically, providing an efficient and responsive user interface.

Navigation within the app is handled using Fragments, allowing seamless switching between the event list and event creation screens. Input validation is also implemented to prevent invalid data entry, such as empty fields or selecting past dates.

Overall, the app demonstrates the use of modern Android development practices, including local database management, structured UI design, and efficient data handling.
